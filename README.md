# Cucumber Blackberry

A low-fatigue dark VS Code theme inspired by the colors of LaCroix Cucumber Blackberry.

## Features

- 🍃 Soft sage green strings
- 🫐 Blackberry purple keywords
- 🌾 Warm cream text
- 🌙 Deep blackberry editor background
- 👀 Comfortable for long programming sessions

## Installation

Install from the VS Code Marketplace (coming soon) or install the `.vsix` manually.
